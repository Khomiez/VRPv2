## ภาพรวมโครงการ

### วัตถุประสงค์

ปัญหา Vehicle Routing Problem (VRP) เป็นปัญหาการวางแผนเส้นทางการเดินทางของยานพาหนะเพื่อให้บริการลูกค้าหลายแห่ง โดยมีจุดเริ่มต้นและสิ้นสุดที่จุดศูนย์กลาง (Depot) โดยมีเป้าหมายเพื่อ:

1. **ลดต้นทุนรวม** (Total Cost) ประกอบด้วยค่าใช้จ่ายคงที่ (ค่ารถ) และค่าใช้จ่ายแปรผัน (ค่าน้ำมัน)
2. **ลดระยะทางรวม** (Total Distance) ที่รถต้องเดินทาง
3. **เพิ่มประสิทธิภาพการใช้งานรถ** (Vehicle Utilization)
4. **รักษาเงื่อนไขความจุ** (Capacity Constraints) ของรถ

### สถานการณ์ปัญหา

```
┌─────────────────────────────────────────────────────────┐
│                   สถานการณ์ปัญหา VRP                   │
├─────────────────────────────────────────────────────────┤
│  🏭 จุดทิ้งขยะ (Depot): 1 จุด                          │
│  🏘️ จุดเก็บขยะ: 20-137 จุด (ตาม Sheet)               │
│  🚛 ยานพาหนะ: รถขยะพร้อมใช้งาน                       │
│  💰 ต้นทุน: ค่ารถคงที่ + ค่าน้ำมัน                  │
│  📦 ความจุ: ขยะทั่วไป 2,000 กก. + รีไซเคิล 200 กก.    │
└─────────────────────────────────────────────────────────┘
```

## อัลกอริทึมที่ใช้

### Guided Local Search (GLS)

อัลกอริทึมที่ใช้คือ **Guided Local Search** ซึ่งเป็นเมตาฮิวริสติก (Metaheuristic) สำหรับแก้ปัญหา VRP โดยใช้ Google OR-Tools Library

#### ขั้นตอนการทำงาน

```
┌─────────────────────────────────────────────────┐
│  STEP 1: เริ่มต้นจากเส้นทางง่ายที่สุด          │
│  (Path Cheapest Arc)                             │
│  → เลือกจุดที่ใกล้ที่สุดเสมอ                         │
├─────────────────────────────────────────────────┤
│  STEP 2: ตรวจสอบข้อจำกัด                         │
│  → ความจุรถ (General + Recycle)                │
│  → ทุกจุดต้องได้รับการเก็บขยะ                    │
│  → เริ่มและสิ้นสุดที่ Depot                        │
├─────────────────────────────────────────────────┤
│  STEP 3: ปรับปรุงเส้นทาง (Guided Local Search)    │
│  → 2-Opt: สลับลำดับ 2 จุด                      │
│  → 3-Opt: สลับลำดับ 3 จุด                      │
│  → Or-Opt: สลับช่วงยาวๆ                         │
├─────────────────────────────────────────────────┤
│  STEP 4: ทำซ้ำจนกดีที่สุด                          │
│  → หมดเวลา (20 วินาที/Sheet)                    │
│  → ไม่ดีขึ้นอีกต่อไป                               │
├─────────────────────────────────────────────────┤
│  STEP 5: แสดงผลเส้นทางที่ดีที่สุด                │
└─────────────────────────────────────────────────┘
```

#### สูตรการคำนวณต้นทุน

```
ต้นทุนรวม = ต้นทุนคงที่ + ต้นทุนแปรผัน

ต้นทุนคงที่:
- ค่ารถ: 2,400 บาท/คัน
- ค่าคนขับ: รวมในค่ารถ
- ค่าซ่อมบำรุง: รวมในค่ารถ

ต้นทุนแปรผัน:
- ระยะทาง × ค่าน้ำมัน (8 บาท/กม.)
```
